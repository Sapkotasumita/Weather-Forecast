# Weather-Forecast 

A simple **weather forecast web app** that displays current weather and forecast information using a weather API. This project was built with HTML, CSS, and JavaScript.

## Features
Shows current weather conditions  
Displays forecast for the next few days  
Looks clean and responsive with basic layout  
Easy to set up and run locally

## Overview
This app fetches weather data from an external API and displays it dynamically on the webpage. Users can enter a city or allow geolocation (if implemented) to get their local weather.

## Built With

- **HTML** – Markup and structure  
- **CSS** – Styling and layout  
- **JavaScript** – Weather API calls and DOM updates
